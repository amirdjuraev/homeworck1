Задание 2

Что такое CMS?


CMS-"Система управления контентом" (движок)-Написанная на PHP-программистами основа для сайта, с помощью которой вы можете управлять сайтом (добавлять контент, менять пункты меню и т.п.) не зная HTML и CSS


Однако, для того чтобы сделать сайт с помощью CMS потребуются услуги программиста, и дизайнера, и верстальщика. И капиталовложения.


Какие бывают CMS?


Бывают различные системы управления контентом: для интернет-магазинов, для блогов, для форумов и т.д.


Примеры CMS

Примеры популярных CMS: Loomla, WordPress (для блогов), PhpBB (для форумов)


СМS-ки бывают платные и бесплатные.

PHP и JavaScript


Языки программирования PHP и JavaScript позволяют сделать сайт динамичным, то есть реагирующим на действия пользователя. Например, можно сделать красивую выпадающую менюшку или слайдер.


Виды скриптов

Для этого пишутся скрипты (англ. script-"сценарий")-программы позволяющие реагировать на действия пользователя. Скрипты бывают двух видов:


  • те которые выполняются на сервере, а результат их выполнения приходит в браузер к пользователю уже в готовом виде. Эти скрипты написанны на языке PHP. На нём пишутся CMS-ки системы управления контентом.
  • те которые выполняются прямо в браузере пользователя. Это скрипты, написанные на языке JavaScript. Они чаще всего используются для, того чтобы сделать станицу более удобной и красивой.
